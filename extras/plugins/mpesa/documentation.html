DOCUMENTATION


M-Pesa Plugin (v1.0) for LaraClassifier and JobClass.

Installation

To install this plugin:
  • Copy the plugin's (uncompressed) folder into the script's plugins folder like this: /extras/plugins/mpesa/
  • Go to your Admin panel → Plugins (and click on "Install") to install it.

Compatibility

  • v1.0 => LaraClassifier (Latest Version) and JobClass (Latest Version)

Update

Replace all the plugin's files on your server.

Configuration

Setting up credentials
  • Get the API key and API secret by going to the M-Pesa developer portal: https://developer.mpesa.vm.co.mz/ and creating an account. Generate an API key and secret from there.
  • Copy and paste the API key and API secret into the /.env file like this:
MPESA_API_KEY=your-mpesa-api-key
MPESA_API_SECRET=your-mpesa-api-secret

Translation

To translate this plugin into your language:
  • Go to the languages folder at /extras/plugins/mpesa/lang/
  • Duplicate the folder /extras/plugins/mpesa/lang/en/ to /extras/plugins/mpesa/lang/<abbr>/
    NOTE: Don't forget to replace <abbr> with your language code (e.g., "de" for the German language).
  • Open the file /extras/plugins/mpesa/lang/<abbr>/messages.php and translate it.
    NOTE: To translate the file, you must change the values and not the keys of the PHP array.

Uninstallation

To uninstall this plugin, go to your Admin panel → Plugins (and click on "Uninstall") to uninstall it.

Removing

To remove this plugin:
  • Uninstall it
  • Go to the folder /extras/plugins/ and remove the folder mpesa/.
